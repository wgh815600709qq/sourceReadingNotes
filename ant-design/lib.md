# libs


> omit.js

* About

Utility function to create a shallow copy of an object which had dropped some fields.

* Link
 [github-link](https://github.com/benjycui/omit.js)


* Usage

```
var omit = require('omit.js');
omit({ name: 'Benjy', age: 18 }, [ 'name' ]); // => { age: 18 }
```
